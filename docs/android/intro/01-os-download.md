# Android 固件下载

## 固件的区别

| 后缀   |                                                              |
| ------ | ------------------------------------------------------------ |
| `eng`  | 开发版，开放串口控制台，ROOT等。内核适配软件调试时使用。     |
| `user` | 用户使用版，关闭串口控制台，加快启动速度，`adb`调试需要授权。产品固件使用。 |
